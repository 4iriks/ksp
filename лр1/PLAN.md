# Практическая работа №1 — Верстка сайта с HTML

## Цель
Разработать примитивный Web-сайт по теме **«Каталог смартфонов»** из нескольких HTML-страниц.

---

## Структура сайта

```
лр1/
├── index.html          — Главная страница
├── catalog.html        — Каталог смартфонов
├── item_iphone.html    — Страница товара: iPhone 15 Pro
├── item_samsung.html   — Страница товара: Samsung Galaxy S24 Ultra
├── item_pixel.html     — Страница товара: Google Pixel 8 Pro
└── PLAN.md             — Этот файл
```

---

## Теги, использованные в работе

| Тег | Назначение |
|-----|-----------|
| `<h1>`, `<h2>`, `<h3>` | Заголовки |
| `<p>` | Параграф |
| `<hr>` | Горизонтальная черта |
| `<ul>`, `<li>` | Маркированный список |
| `<a href="">` | Ссылка (относительная и абсолютная) |
| `<img>` | Изображение |
| `<b>`, `<i>` | Жирный, курсив |
| `<small>` | Уменьшенный текст |
